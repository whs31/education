–°–ª–æ–≤–∞—Ä—å - —ç—Ç–æ **–∞—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω—ã–π –º–∞—Å—Å–∏–≤**. –≠—Ç–æ –∑–Ω–∞—á–∏—Ç, —á—Ç–æ –≤ —Å–ª–æ–≤–∞—Ä–µ –¥–∞–Ω–Ω—ã–µ —Ö—Ä–∞–Ω—è—Ç—Å—è –ø–æ–ø–∞—Ä–Ω–æ: –∫–ª—é—á –∏ –∑–Ω–∞—á–µ–Ω–∏–µ.
–ü—Ä–∏–º–µ—Ä: 
- –¢–µ–ª–µ—Ñ–æ–Ω–Ω–∞—è –∫–Ω–∏–≥–∞ - —ç—Ç–æ —Å–ª–æ–≤–∞—Ä—å: –∫–ª—é—á - –∏–º—è —á–µ–ª–æ–≤–µ–∫–∞ (`str`), –∑–Ω–∞—á–µ–Ω–∏–µ - –µ–≥–æ —Ç–µ–ª–µ—Ñ–æ–Ω (`str`)
- –ú–µ–Ω—é –≤ —Ä–µ—Å—Ç–æ—Ä–∞–Ω–µ - —ç—Ç–æ —Å–ª–æ–≤–∞—Ä—å: –∫–ª—é—á - –Ω–∞–∑–≤–∞–Ω–∏–µ –±–ª—é–¥–∞ (`str`), –∑–Ω–∞—á–µ–Ω–∏–µ - –µ–≥–æ —Ü–µ–Ω–∞ (`int`)

–û–±—ã—á–Ω–æ –≤ —Å–ª–æ–≤–∞—Ä–µ –∫–ª—é—á–µ–º –≤—ã—Å—Ç—É–ø–∞–µ—Ç —Å—Ç—Ä–æ–∫–∞ (`str`) –∏–ª–∏ –∂–µ —á–∏—Å–ª–æ (`int`). –ó–Ω–∞—á–µ–Ω–∏–µ–º –º–æ–∂–µ—Ç –±—ã—Ç—å —á—Ç–æ —É–≥–æ–¥–Ω–æ - –Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–ø–∏—Å–æ–∫ `list` –∏–ª–∏ –∂–µ –¥—Ä—É–≥–æ–π —Å–ª–æ–≤–∞—Ä—å `dict`.

–°–ª–æ–≤–∞—Ä–∏ –≤ —è–∑—ã–∫–µ Python –æ–±—ä—è–≤–ª—è—é—Ç—Å—è —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –∫–ª—é—á–µ–≤–æ–≥–æ —Å–ª–æ–≤–∞ `dict` –∏–ª–∏ —Ñ–∏–≥—É—Ä–Ω—ã—Ö —Å–∫–æ–±–æ–∫ `{}`.

–î–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –∏–∑ —Å–ª–æ–≤–∞—Ä—è –∏–ª–∏ –∑–∞–ø–∏—Å–∏ –≤ –Ω–∏—Ö –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Ç–µ –∂–µ –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã–µ —Å–∫–æ–±–∫–∏, —á—Ç–æ –∏ –≤ —Å–ø–∏—Å–∫–∞—Ö (`[]`). –û–¥–Ω–∞–∫–æ, –≤–º–µ—Å—Ç–æ –∏–Ω–¥–µ–∫—Å–∞ —ç–ª–µ–º–µ–Ω—Ç–∞, –≤ —Å–ª–æ–≤–∞—Ä–µ –≤ –Ω–∏—Ö –ø–µ—Ä–µ–¥–∞–µ—Ç—Å—è *–∫–ª—é—á*, –Ω–∞–ø—Ä–∏–º–µ—Ä, —Å—Ç—Ä–æ–∫–∞.

–ü—Ä–∏–º–µ—Ä - –æ–±—ä—è–≤–∏–º —Å–ª–æ–≤–∞—Ä—å —Å –ø—Ä–æ—Å—Ç–æ–π —Å—Ç—Ä—É–∫—Ç—É—Ä–æ–π:
- –ö–ª—é—á $key$ - –∏–º—è —Å—Ç—É–¥–µ–Ω—Ç–∞
- –ó–Ω–∞—á–µ–Ω–∏–µ $value$ - –µ–≥–æ –≤–æ–∑—Ä–∞—Å—Ç
```python
students = dict()

# –¥–æ–±–∞–≤–∏—Ç—å –≤ —Å–ª–æ–≤–∞—Ä—å –∑–Ω–∞—á–µ–Ω–∏–µ 
students['Alexa'] = 18
```

#### –ú–µ—Ç–æ–¥—ã —É —Å–ª–æ–≤–∞—Ä—è
###### –û—á–∏—Å—Ç–∏—Ç—å —Å–ª–æ–≤–∞—Ä—å:
–î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º –º–µ—Ç–æ–¥ `clear()`:
```python
drinks = {
	'coffee': 'tasty',
	'tea': 'tasty',
	'water': 'tasty',
	'cola': 'tasty',
	'sprite': 'not tasty =('
}

print(drinks) # –≤—ã–≤–µ–¥–µ—Ç –ø—É—Å—Ç–æ–π —Å–ª–æ–≤–∞—Ä—å - {}
```

###### –ü–æ–ª—É—á–∏—Ç—å –≤—Å–µ –∫–ª—é—á–∏ –∏ –ø–æ–ª—É—á–∏—Ç—å –ø–∞—Ä—ã –∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ:
–ó–∞—á–∞—Å—Ç—É—é –ø–æ —Å–ª–æ–≤–∞—Ä—é –Ω—É–∂–Ω–æ –∏—Ç–µ—Ä–∏—Ä–æ–≤–∞—Ç—å—Å—è —á–µ—Ä–µ–∑ —Ü–∏–∫–ª. –î–ª—è —ç—Ç–æ–≥–æ –º—ã –º–æ–∂–µ–º –ø–æ–ª—É—á–∏—Ç—å –≤—Å–µ –∫–ª—é—á–∏ –≤ —Å–ª–æ–≤–∞—Ä–µ - –º–µ—Ç–æ–¥ `keys()`, –∏ –≤—Å–µ –ø–∞—Ä—ã –∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ - –º–µ—Ç–æ–¥ `items()`. –û–±–∞ —ç—Ç–∏—Ö –º–µ—Ç–æ–¥–∞ –≤–µ—Ä–Ω—É—Ç —Å–ø–∏—Å–∫–∏:
```python
drinks = {
	'coffee': 'tasty',
	'tea': 'tasty',
	'water': 'tasty',
	'cola': 'tasty',
	'sprite': 'not tasty =('
}

print(drinks.keys())    # –≤—ã–≤–µ–¥–µ—Ç –≤—Å–µ –∫–ª—é—á–∏
print(drinks.items())   # –≤—ã–≤–µ–¥–µ—Ç –≤—Å–µ –∫–ª—é—á–∏ –≤ –ø–∞—Ä–µ —Å–æ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏

# –≤—ã–≤–µ–¥–µ–º –ø–∞—Ä—É –∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ, –µ—Å–ª–∏ –∏–º—è –Ω–∞–ø–∏—Ç–∫–∞ –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è –Ω–∞ c:
for drink in drinks.keys():
	if drink.startswith('c'):
		print(drink, end=" ") 

print()

# –≤—ã–≤–µ–¥–µ–º –∏–º—è –Ω–∞–ø–∏—Ç–∫–∞, –µ—Å–ª–∏ –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏–µ - –Ω–µ tasty:
for drink, tastiness in drinks.items(): 
	if tastiness != 'tasty':
		print(f'Drink {drink} is not tasty!')
```
–í—ã–≤–æ–¥:

###### –£–¥–∞–ª–∏—Ç—å –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç –∏–∑ —Å–ª–æ–≤–∞—Ä—è:
–î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º –º–µ—Ç–æ–¥ `pop(...)`
```python
prices = {
	'eggs': 10.0,
	'milk': 5.0,
	'butter': 7.0,
	'water': 2.0,
	'tea': 4.0
}

# –∑–∞–ø–∏—à–µ–º –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –¥–µ—à–µ–≤–ª–µ 6.0 –≤ –Ω–æ–≤—ã–π —Å–ª–æ–≤–∞—Ä—å
cheap = dict()
for key, price in prices.items():
	if price <= 6:
		cheap[key] = price

# —É–¥–∞–ª–∏–º –∏–∑ —Å—Ç–∞—Ä–æ–≥–æ —Å–ª–æ–≤–∞—Ä—è –≤—Å–µ, —á—Ç–æ –¥–µ—à–µ–≤–ª–µ 6.0

	prices.pop(key)

```


###### –ù–∞–π—Ç–∏, —Å–∫–æ–ª—å–∫–æ —Ä–∞–∑ —Ä–∞–∑–Ω—ã–µ –±—É–∫–≤—ã –≤—Å—Ç—Ä–µ—á–∞—é—Ç—Å—è –≤ —Ç–µ–∫—Å—Ç–µ:
```python
text = '„ÅÑ„Çç„ÅØ„Å´„Åª„Å∏„Å® „Å°„Çä„Å¨„Çã„Çí „Çè„Åã„Çà„Åü„Çå„Åù „Å§„Å≠„Å™„Çâ„ÇÄ „ÅÜ„Çê„ÅÆ„Åä„Åè„ÇÑ„Åæ „Åë„Åµ„Åì„Åà„Å¶ „ÅÇ„Åï„Åç„ÇÜ„ÇÅ„Åø„Åó „Çë„Å≤„ÇÇ„Åõ„Åô'
```

###### –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å —Å—Ç—Ä–æ–∫—É –≤ —Å–ª–æ–≤–∞—Ä—å:
–î–∞–Ω–∞ —Å—Ç—Ä–æ–∫–∞ –∏–∑ –Ω–∞–∑–≤–∞–Ω–∏–π –±–ª—é–¥ –∏ —ç–º–æ–¥–∑–∏. –ù—É–∂–Ω–æ –ø–æ–º–µ—Å—Ç–∏—Ç—å –≤—Å–µ —ç—Ç–æ –≤ —Å–ª–æ–≤–∞—Ä—å —Å –∫–ª—é—á–µ–º - –Ω–∞–∑–≤–∞–Ω–∏–µ–º –±–ª—é–¥–∞ –∏ –∑–Ω–∞—á–µ–Ω–∏–µ–º - —ç–º–æ–¥–∑–∏:
